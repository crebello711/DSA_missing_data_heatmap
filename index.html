CTOT-08 DSA Missing Samples

The purpose of this project was to understand how many samples we require to do DSA testing.


After assembling the dataset, I used seaborn function from python to generate a Heatmap. Factors I looked at included biopsy related visit types and DSA data missing for such visit type from previously collected data. This site provides the data and visualizations created as part of the project, as well as explanations and descriptions of any differences witnessed.
